== Chapter 1: Getting Started

AsciiDoc is a lightweight markup language for creating documentation, articles, books, and more.

=== 1.1 Basic Syntax

Here are some basic AsciiDoc syntax examples:

* *Bold text*
* _Italic text_
* `Monospace text`
